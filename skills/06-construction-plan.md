# 施工方案与施工图设计技能

## 触发条件
当用户提到"施工方案"、"施工组织设计"、"施工图"、"设计方案"、"建井"、"钻探"时自动激活。

## 一、地下水调查施工方案框架

```
1 工程概况
  1.1 项目名称、地点、建设单位
  1.2 工程规模与主要工作量
  1.3 工期要求

2 施工部署
  2.1 施工组织机构
  2.2 施工顺序与流水段划分
  2.3 施工总平面布置

3 施工方案
  3.1 场地平整与道路修筑
  3.2 监测井钻探施工
     - 钻探方法选择（回转钻/冲击钻/空气潜孔锤）
     - 钻孔结构设计（孔径、井管、滤水管、填砾、止水）
     - 地层录井与取样
     - 成井工艺（下管、填砾、止水、洗井）
  3.3 土壤/地下水采样
     - 采样方法（低流量采样/微洗井采样/贝勒管）
     - 质控措施（平行样、运输空白、现场空白）
     - 样品保存与运输
  3.4 水文地质试验
     - 抽水试验
     - 注水试验
     - 示踪试验

4 质量保证措施
  4.1 钻探质量控制要点
  4.2 采样质量控制要点
  4.3 关键工序验收标准

5 安全文明施工
  5.1 安全管理制度
  5.2 主要危险源及应对
  5.3 应急预案
  5.4 文明施工与环保措施

6 进度计划
  6.1 总体进度计划（甘特图）
  6.2 关键线路分析
  6.3 工期保障措施

7 资源配置
  7.1 劳动力配置
  7.2 机械设备配置
  7.3 主要材料需用量计划
```

## 二、施工图设计要点

### 监测井结构图设计要素
- 地层柱状图（按实际钻探地层）
- 井管结构（实管段、滤水管段、沉砂管段）
- 井周填料分层（滤料层、止水层、回填层）
- 井口保护装置（井台、井盖、锁具）
- 标注关键尺寸和高程

### 监测井平面布置图设计原则
- 覆盖所有已识别和疑似污染源
- 沿地下水流向布设（上游对照→源区→下游扩散区）
- 垂直于地下水流向布设截面线
- 不同含水层分层监测
- 标注坐标系（建议2000国家大地坐标系）

### 图面要求
- 比例尺标注
- 图例完整
- 坐标网格
- 指北针
- 责任边界线
- 企业/设施位置标注
- 现有井和新建井用不同符号区分

## 三、地下水修复工程施工方案要点

### 常见修复技术施工要点

#### 抽出处理（P&T）
- 抽水井布设与抽水量设计
- 地面水处理站工艺设计
- 回灌井设计（如需）
- 运行参数与调试方案

#### 原位化学氧化（ISCO）
- 注入井网格设计
- 药剂选择与配比（高锰酸钾/过硫酸盐/芬顿）
- 注入量计算与注入速率控制
- 安全防护措施（放热反应/有毒气体）

#### 可渗透反应墙（PRB）
- 基槽开挖方案
- 反应介质选择（零价铁/活性炭/生物介质）
- 填充与压实工艺
- 水力学设计验证

#### 监控自然衰减（MNA）
- 监测网络优化
- 衰减速率评估方法
- 长期监测方案设计
