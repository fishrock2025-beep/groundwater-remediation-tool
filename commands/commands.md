# 斜杠命令定义

## /gw:data-review — 实验室数据审查
**用途**：上传检测数据后，自动执行质控审查 + 超标评价 + 时序分析
**工作流**：
1. 确认评价标准（GB/T 14848 哪一类）
2. 检查检测方法适用性
3. 统计超标率、超标倍数、最大值
4. 按责任边界分区统计
5. 识别时序变化趋势
6. 标注异常值和可能原因

## /gw:source-trace — 污染源溯源分析
**用途**：根据企业原辅料清单和检测数据，进行污染源-污染物匹配分析
**工作流**：
1. 梳理原辅料中的目标污染物
2. 与地下水检出物进行匹配
3. 分析空间关联性
4. 给出各污染源的贡献可能性排序

## /gw:report-review — 报告审核
**用途**：审核调查报告/分析报告的逻辑严密性
**工作流**：
1. 检查评价标准选用
2. 检查数据完整性和质控
3. 检查原因分析的逻辑链条
4. 识别推理漏洞和证据缺口
5. 给出修改建议

## /gw:brief-draft — 汇报材料起草
**用途**：将技术报告转化为政府汇报材料
**输入**：技术报告 + 汇报对象级别
**工作流**：
1. 提炼核心数据（不超过5个关键数字）
2. 简化原因分析为通俗表述
3. 列出已采取和拟采取措施
4. 控制篇幅
5. 调整口径和用词

## /gw:bid-outline — 投标文件大纲
**用途**：根据招标文件生成技术标大纲和重难点分析
**工作流**：
1. 解读招标文件核心需求
2. 生成响应性技术标结构
3. 分析项目重点难点
4. 建议加分亮点

## /gw:construction — 施工方案编写
**用途**：编写地下水调查或修复工程的施工方案
**工作流**：
1. 确认工程类型（调查/修复/监测）
2. 生成方案框架
3. 填充技术参数
4. 编写安全文明施工章节

## /gw:well-design — 监测井设计
**用途**：根据水文地质条件设计监测井结构
**输入**：目标层位、深度、含水层类型
**输出**：井结构设计参数 + 结构示意图描述
